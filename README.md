Requirements
------------

- PHP 5.5 or higher

- [Nette Framework](https://github.com/nette/nette) 2.2



Installation
------------

The best way to install Instante UI is using  [Composer](http://getcomposer.org/):

```sh
$ composer require instante/ui
```
